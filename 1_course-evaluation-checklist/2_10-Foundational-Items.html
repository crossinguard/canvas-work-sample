10 Foundational Items

While the entire Course Evaluation Checklist v2.0 contains nearly 40 items, these 10 Foundational Items can serve as a quick start guide for improving a Canvas course. Future modules will explore each of the foundational items across the four areas of focus, along with all accompanying items.

1. Course Information

  • 1.1 | The Home Page (i.e., Front Page) should provide a visual representation of the course, including a brief course description or introduction, clear instructions for students on where to begin, and navigation to easily access current content.
  • 1.2 | The Course Navigation should be clear, consistent, and uncluttered, with unused navigation items hidden.
  • 1.3 | Key Learning Information (e.g., goals, learning objectives, content standards) should be provided with periodic or persistent reminders.
  • 1.4 | Class Expectations (e.g., participation, etiquette, code of conduct), Policies (e.g., grading, late work, make-up work), and Technology Requirements should be provided and clearly defined.
  • 1.5 | Instructor Contact Information should be made available to students, including availability, communication preferences, expected response time, with an optional biography and picture offering personalization.

2. Course Content

  • 2.1 | Copyright should be honored.
  • 2.2 | All Links, Files, Videos, and URLs should be active and functional.

3. Assessment of Student Learning

  • 3.1 | Sufficiently detailed Instructions should be provided to ensure clarity of expected responses within an assessment activity.
  • 3.2 | A Variety of Assessment Types are used (e.g., discussions, quizzes, individual/group assignments).

4. Course Accessibility

  • 4.1 | Minimize Accessibility Issues by employing web tools or software such as the built-in Accessibility Checker.
